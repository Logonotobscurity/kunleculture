# **App Name**: Kunle Couture Replica

## Core Features:

- Navbar: Implements a fixed navbar with logo, links (Shop, Bespoke, Lookbook, About), Login button, and Menu button.
- Hero Section: Showcases full-bleed image/video, animated rotating categories (Bespoke Agbadas, Tailoring Services, etc.), and CTAs.
- Category Grid: Displays primary cards (Bespoke, Collections, Tailoring, Made-to-Measure) and secondary compact cards (Accessories, Lookbook, Visit Studio) with image backgrounds, badges, and hover effects.
- Collection Sections: Features seasonal collection heros with gradient overlays, collection cards with quick view and add-to-wishlist, and bespoke & tailoring sections with timelines and feature cards.
- Editorial Section: Displays a carousel of editorial imagery, video cards (campaign films), and style guides.
- Craft and Knowledge Section: Illustrates construction techniques, fabric guides, care instructions, and editorial posts on style and heritage.
- Newsletter Section: Includes an email input, subscribe button, and booking studio appointments.

## Style Guidelines:

- Primary color: Teal (#0B4F4F), aligning with the brand's sophisticated and modern aesthetic.
- Background color: Desaturated teal (#F5F5F4), provides a clean and refined base.
- Accent color: Pink (#831843), contrasting color used sparingly to highlight key elements and CTAs.
- Body and headline font: 'Sofia Pro', a sans-serif font to emphasize the blend of tradition and modernity.
- Lucide React Icons for consistent UI elements
- Base unit of 4px spacing system and mobile-first design to ensure a responsive experience across various devices.
